ABOUT

라온시큐어에서 근무를 시작했으며, 모바일 백신 솔루션 TouchEn mVaccine, 모바일 백신 서비스 라온 모바일 시큐리티, 웹 가상 키패드 TouchEn Transkey 개발을 담당했습니다.

TouchEn mVaccine 솔루션은 하나금융그룹을 포함한 30개 이상의 금융그룹에서 사용 중이며, 구글 플레이 스토어에서 1000만 이상이 다운로드 하였습니다.

라온 모바일 시큐리티는 구글 플레이 스토어에서 50만명 이상이 다운로드 하여 평점 4.2점으로 1000명 이상이 긍정적인 평가를 남겼습니다.

TouchEn Transkey는 금융기관, 공공기관에서 많이 사용하며, 교육청에서는 매일 동시 접속 500만 이상에게 안정적인 서비스를 제공을 했습니다.

TouchEn Transkey / mTranskey

가상 키보드 이용해 정보 입력을 보안해주는 웹 어플리케이션 솔루션입니다. 매번 바뀌는 키 위치, 암호화된 좌표, 멀티 마우스 커서를 지원하여 입력 값을 보안해줍니다. 기기, 플랫폼에 구애받지 않는 완전한 반응형 웹 디자인을 제공합니다. 웹 접근성 심사기준을 완벽하게 준수합니다.

- Custom UI 개발
- 반응형 웹 디자인
- 세션 미사용 개발
- 패턴 잠금 개발
- 웹 접근성 준수
- REST API 제공

TouchEn nxKey

- 키 입력 Hooking Daemon 개발
